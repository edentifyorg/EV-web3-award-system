# NEVERFLAT Demo UI

This frontend is a Vite + React + TypeScript demo wallet UI for the NEVERFLAT award backend.

## Commands

```bash
npm install
npm run dev
```

Dev server runs on `http://localhost:3001`.

Build for production:

```bash
npm run build
npm run preview
```

## Backend connection

The UI calls the backend API (default `http://localhost:3000`) and supports `X-API-Key` for protected endpoints.

Run backend and frontend in parallel for local demo.
